Introduction

Bienvenue sur le tutoriel d'utilisation de la plateforme. Ce tutoriel vous guidera à travers les principales fonctionnalités et vous aidera à naviguer efficacement.

Navigation

Utilisez le menu de navigation en haut de la page d'acceuil pour accéder aux différentes sections du site. Pour revenira l'accueil, il vous suffit de cliquer sur le bouton home

Fonctionnalités

Notre site offre plusieurs fonctionnalités pour améliorer votre expérience :

  • Création de compte
  • Ajout d'un appareil utilisateur
  • Visualisation des données en temps réel
  • Téléchargement des données
  • Carte interractive
  • Utilistaion de l'api

Création de compte

Pour créer un compte, cliquez sur le bouton "register" en haut à droite de la page d'accueil. Remplissez les informations requises et suivez les instructions pour terminer la création de votre compte.

Ajout d'un appareil utilisateur

Pour ajouter un appareil utilisateur, cliquez sur le bouton "register device" dans le menu de navigation. Remplissez les informations requises et suivez les instructions pour terminer l'ajout de votre appareil. Vous pouvez alors verifier que l'appreil a bien été ajouté dans la section device list.

Visualisation des données en temps réel

Pour visualiser les données en temps réel, cliquez sur le bouton "Visualize Real-time Data" dans le menu de navigation. Vous pouvez alors voir les données en temps réel de votre appareil.

Téléchargement des données

Pour télécharger les données, cliquez sur le bouton "Download Data" dans le menu de navigation. Vous pouvez alors télécharger les données de votre choix de votre appareil. un second bouton vous permet aussi de télécharger l'intégralité des données

Carte interractive

Pour visualiser les données sur une carte, cliquez sur le bouton "Map" dans le menu de navigation. Vous pouvez alors voir les différents appareils proches ainsi que les objets qu'ils détectent.

Utilisation de l'api

Pour utiliser l'api, rendez vous sur la page de documentation de l'api. Vous y trouverez toutes les informations nécessaires pour utiliser l'api de la plateforme. afin de connaitre votre cé api, rendez vous sur votre page de profil , vous pourrez alors génerer votre propre clé (attention, générer une nouvelle clé annule la précédente)
